# Story 1.3 : Immersive Hero Section
**Epic :** 1. Core Experience & Studio UI

## User Story
En tant que visiteur, je veux arriver sur une page d'accueil visuellement marquante qui définit immédiatement l'univers "Studio".

## Acceptance Criteria
1. La section d'accueil occupe 100% de la hauteur du viewport (100vh).
2. Une image de fond (photo de l'auteur) est présente avec une optimisation `next/image`.
3. Un titre percutant est affiché avec une animation d'entrée fluide via Framer Motion.
