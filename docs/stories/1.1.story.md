# Story 1.1 : Project Setup & Base Architecture
**Epic :** 1. Core Experience & Studio UI

## User Story
En tant que développeur, je veux initialiser le projet avec la stack technique définie afin de poser des bases solides pour l'univers "Studio".

## Acceptance Criteria
1. Le projet Next.js (App Router) est initialisé avec TypeScript et Tailwind CSS.
2. L'arborescence des dossiers suit les recommandations de Winston (Architecte).
3. La configuration de Tailwind inclut la palette de couleurs "Dark Studio".

## Tasks
- [ ] Initialiser l'application Next.js : `npx create-next-app@latest . --typescript --tailwind --eslint`.
- [ ] Configurer `tailwind.config.ts` avec les couleurs sombres personnalisées.
- [ ] Créer les dossiers de structure : `/components`, `/hooks`, `/store`, `/public/audio`.
- [ ] Installer Zustand pour la gestion d'état globale : `npm install zustand`.
- [ ] Installer Framer Motion pour les animations : `npm install framer-motion`.

## Dev Notes
- Utiliser `next/font` pour importer des typographies percutantes dès le départ.
